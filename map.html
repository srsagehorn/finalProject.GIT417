.map() function tutorial

The .map() method is similar to the .forEach() method in that it is a javascript array method, except that it creates a new array populated with the results of whatever function the user inputs.

Why use the .map() method?

For this example we will be working with the following array 

var dogs = ["pitbull", "yorkie", "shitzu", "dalmatian"]

If we attempted to make everything uppercase within the array using a for loop, we would: 

var dogs = ["pitbull", "yorkie", "shitzu", "dalmatian"]
var upperCaseDogs = []
for ( var i = 0; i < dogs.length; i++) {
          upperCaseDogs.push(dogs[i].toUpperCase())
          }
        console.log(upperCaseDogs)
["PITBULL", "YORKIE", "SHITZU", "DALMATIAN"]

Syntax

In the .map() method, you initialize the array you want to work with, then assign a variable to an empty array or the entire .map() statement like you will see in these examples. Then assign the empty array to the initial array followed by .map(). Within those parentheses you can place any function and the method goes through each index in the array and applies it, then places it into the new array.

Function

var array = ["index1", "index2", "index3"]
let newArray = array.map(function (element) {
                your function here
              })

Fat arrow function

var array = ["index1", "index2", "index3"]
let newArray = array.map(element => {your function here})

So for our dogs example the map function would look like:

var dogs = ["pitbull", "yorkie", "shitzu", "dalmatian"]
let uppercaseDogs = dogs.map(dog => dog.toUpperCase())
["PITBULL", "YORKIE", "SHITZU", "DALMATIAN"]

Examples

Here we multiple every number by 2:

numbers = [3, 5, 10, 9]
let doubled = numbers.map(num => num*2)
console.log(doubled)
[6, 10, 20, 18]

Here we change every string to uppercase:

var days = ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"]
let uppercaseDays = days.map(day => day.toUpperCase())
console.log(uppercaseDays)
["MONDAY", "TUESDAY", "WEDNESDAY", "THURSDAY", "FRIDAY", "SATURDAY", "SUNDAY"]
