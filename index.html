.forEach() function tutorial

The forEach() method is a javascript array method. It serves as an alternative to a for loop by calling each element in an array and executing the proceeding function block.

Why use the .forEach() method?

For this example we will be working with the following array 

var fruits = ["apple", "orange", "banana", "cucumber", "strawberry"]

If we attempted to do a for loop, and console log each value it would look like this: 

var fruits = ["apple", "orange", "banana", "cucumber", "strawberry"]
for ( var i = 0; i < fruits.length; i++) {
            console.log(fruits[i])
          }

orange
banana

Syntax

In the .forEach() method, you initialize the array you want to work with, then call .forEac() directly after the array name in a statement. Within those parentheses you can place any function and the method goes through each index in the array and applies that function. Unlike the .map() method, it will not place it back into an array.

Function

var array = ["index1", "index2", "index3"]
array.forEach(function (element) {
            your function here
            })

Fat arrow function

var array = ["index1", "index2", "index3"]
array.forEach(element => your function here)

So for our fruits example the forEach function would look like: 

var fruits = ["apple", "orange", "banana", "cucumber", "strawberry"]
fruits.forEach(fruit => console.log(fruit))
apple
orange
banana
cucumber
strawberry
